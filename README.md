#Projekt ToDoTaskAPP 

##Uruchomienie
Aby uruchomić projekt należy uruchomić dockera komendą
```
docker-compose up --build
```
Po uruchomieniu kontenerów można odpalić projekt WPF


##Użyte technologie:
.NET 6.0 WEB API

SQLServer 2019

RabbitMQ

Docker

EntityFramework

Swagger
